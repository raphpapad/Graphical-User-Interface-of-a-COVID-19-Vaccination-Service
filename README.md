Design and evaluate a Graphical User Interface of a COVID 19 Vaccination Service.

Overview:
- Phase A: Interactive User Interface Design (Lo Fi & Hi Fi prototyping)
- Phase B: User based Evaluation
- Working with Figma

Primary Functions:
- Make a Reservation
- Fill in Personal Details
- Fill in Escort Details
- Fill in Health Profile
- Preview Reservation
- Modify/Cancel Current Reservation
- View Vaccination Certificate

Phase A: Interactive UI Design
- Low Fidelity prototypes
- High Fidelity prototypes
- Design tool: Figma

Phase B: Evaluation
- Conduct evaluation sessions with real users
   1. Pilot testing
   2. Real users
   3. Write report
